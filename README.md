## ğŸš€ How to Use

### Setup
1. Extract all files to your web server directory
2. Ensure all files are in the correct folder structure
3. No build process required - ready to go!

### Customization
To customize the website:
1. **Colors**: Edit CSS variables in `css/styles.css` (`:root` section)
2. **Content**: Edit HTML in respective page files
3. **Images**: Replace placeholder background colors with actual images
4. **Forms**: Configure form submission endpoints
5. **Links**: Update social media and external links

### Adding Content
1. **News Articles**: Add new `<article class="card">` elements in `pages/news.html`
2. **Gallery Images**: Add `<div class="gallery-item">` in `pages/gallery.html`
3. **Alumni**: Update table/cards in `pages/directory.html`
4. **Events**: Add to event listing in `pages/reunions.html`

## ğŸ¨ Color Customization

```css
:root {
    --primary-blue: #003d7a;      /* Dark blue */

    --light-accent: #e7f0ff;      /* Very light blue */
```
